 RockJS Framework transforma sus Sistemas Legados

RockJS reutiliza la arquitectura legada y permite:

Exponer sus Sistemas Legados en navegadores Web

Sin reescribir lógica de negocio. La interfaz de sus Sistemas Legados transformadas a Web, y totalmente funcionales sobre navegadores estándar

Expone e intercambia información mediante Servicios Web Síncronos

Con protocolos REST (JSON) o (XML)

Transacciones con apps nativas.

Interactúe en línea con clientes, fuerza de ventas, proveedores y cualquier otro dispositivo.

Acceda desde cualquier navegador web

RockJS Framework es la mejor opción para eliminar las barreras tecnológicas entre sus Sistemas Legados y las nuevas tendencias tecnológicas.

Instale en cualquier Sistema Operativo
